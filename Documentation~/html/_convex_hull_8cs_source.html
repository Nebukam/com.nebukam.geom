30 //Generate a counter-clockwise convex hull with the jarvis march algorithm (gift wrapping)
31 //The algorithm is O(n*n) but is often faster if the number of points on the hull is fewer than all points

33 //Is more robust than other algorithms because it will handle colinear points with ease

49 //These might not be ccw, and they may also be colinear

64 //If several have the same x coordinate, find the one with the smallest z
